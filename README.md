# είλως - Typescript Preset

This preset can be used for a typical, modern Typescript project with React.

## Components

* Webpack (4.43)
* Typescript (3.9)
* React (16.9)
* Jest (26.0)

## Supported Lifecycle Steps

### `build`

```js
{
  ...
  "scripts": {
    "build": "eilos build"
  }
  ...
}
````
